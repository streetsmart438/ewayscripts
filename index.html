Key Features

License Verification

Automatically checks player driving licenses when entering vehicles as the driver.

Access Denied Messages

Clear notifications inform players when they lack proper licensing to operate a vehicle.

Police Alerts

Automatic notifications to law enforcement with location and vehicle details for unlicensed drivers.

Job Exemptions

Police, ambulance, and mechanic jobs can operate vehicles without standard licensing requirements.

Grace Period

3-second warning period before enforcement actions, allowing voluntary compliance.

Admin Commands

Full administrative control to grant, revoke, or check license status for any player.

Installation Instructions

Download the Resource
Place the qb-drivinglicense folder in your FiveM server's resources directory.
Update Server Configuration
Add the following line to your server.cfg:
ensure qb-drivinglicense
Configure Settings
Edit config.lua to customize vehicle classes, job exemptions, and alert settings.
Restart Server
Restart your FiveM server to load the new resource.

Admin Commands

/givelicense [player_id] - Grant driving license to a player
/removelicense [player_id] - Remove driving license from a player
/checklicense [player_id] - Check player's license status
Note: These commands require admin permissions and are only available to authorized server administrators.
